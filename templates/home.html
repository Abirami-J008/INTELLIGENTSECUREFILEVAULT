🔐 Core Security Features

Designed with enterprise-grade security architecture

Role Based Access

Separate Admin and User access. Users can only access files belonging to their department.

Password Encryption

Passwords are securely protected using bcrypt hashing before being stored in database.

File Encryption

All uploaded files are encrypted using Fernet Encryption before being stored on server.

🚨 Advanced Security Protection

Real-time protection against unauthorized access

Failed Login Detection

System continuously monitors incorrect password attempts to identify suspicious activity.

3 Attempt Protection

After three incorrect password attempts, account protection activates automatically.

10 Minute Lock

Account becomes temporarily locked for 10 minutes to prevent brute force attacks.
